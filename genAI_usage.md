# Generative AI Usage

## Overview
Generative AI tools (such as ChatGPT) were used during this project to assist with development, debugging, and report writing.

---

## Example 1: Code Assistance
**Prompt:**
How to deploy a machine learning model on EC2?

**Response Summary:**
Provided steps for deploying a model using EC2 and suggested using external storage instead of GitHub for large files.

---

## Example 2: Debugging
**Prompt:**
Why is my model file too large for GitHub?

**Response Summary:**
Explained GitHub file size limitations and suggested alternative solutions such as cloud deployment.

---

## Example 3: Report Writing
**Prompt:**
Help me structure a software engineering report.

**Response Summary:**
Generated a structured report template with sections such as requirements, architecture, and testing.

---

## Reflection
The use of generative AI significantly improved productivity by providing guidance and suggestions. However, all outputs were reviewed and adapted to fit the project requirements.
